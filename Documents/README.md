# Life Engine Documents

This folder contains implementation-facing documentation for platform support, logging behavior, and engine runtime invariants.

## Documents

- `PlatformSupport.md` - supported host and target platforms, Windows/Linux/macOS architecture notes, and CI/build expectations.
- `Logging.md` - engine logging configuration, threading characteristics, and recommended usage.
- `EventThreadingInvariants.md` - event flow, runtime ownership, and thread-safety boundaries for the current engine architecture.
